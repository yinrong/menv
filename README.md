# menv
Inheritable Multiple Environment Configuration for Any Language

![Demo: menv build](https://github.com/yinrong/menv/raw/master/demo.png "menv build")

# Usage

## Prepare Config

example: https://github.com/yinrong/menv/tree/master/test/conf


## Build Config
```sh
menv build
```

## Echo Config
```sh
menv echo <env> <path>
```
